# Med Reminder — GitHub Pages

Static legal and support pages for the Med Reminder iOS app.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Landing / links |
| `privacy.html` | Privacy Policy |
| `terms.html` | Terms of Use |
| `support.html` | Support |
| `style.css` | Shared styles |

## Enable GitHub Pages

1. Repository **Settings** → **Pages**
2. **Build and deployment** → Source: **Deploy from a branch**
3. Branch: **main** → folder: **/ (root)**
4. Save. Site URL: `https://salefurkan.github.io/medication-tracking-pages/`

After deploy, update app `LegalURLs` if the public URL changes from the legacy `furkansale.github.io/ilac-takip-hatirlatici` paths.
